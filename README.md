## 🎯 Purpose

Learn systematic approach to troubleshooting infrastructure issues:
- Identify problem clearly
- Investigate methodically  
- Find root cause
- Apply solution
- Verify & document

## 📚 How to Use

Each case documented with:
- Problem statement
- Investigation steps (with commands)
- Root cause analysis
- Solution applied
- Key learnings

## 🛠️ Tools Used

- Linux command line (netstat, systemctl, ufw, top, etc)
- VirtualBox Ubuntu Server
- System monitoring tools
